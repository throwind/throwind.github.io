## 💡 为什么要做安全配置？

打个比方：OpenClaw 就像是一个不知疲倦的**超级管家**，他能帮你读写电脑里的文件、运行程序、甚至帮你发微信、Telegram。

- **如果不做配置**：等于大门敞开，黑客或者陌生人可以通过聊天软件直接对你的电脑下达"格式化硬盘"或"窃取密码"的指令，管家会毫无防备地照做。
- **做完最新版配置后**：管家只听你一个人的话；陌生人找他必须经过你点头同意；管家被关进了一个没有任何后门的"绝对安全屋"（隔离沙箱）里；并且禁止管家自己"无限分身"（禁止无限生成子代理）。

## 🚀 第一步：使用"一键提示词"完成配置

请直接**复制下面方框里的所有文字（提示词）**，发送给你的 AI 编程助手（比如现在的 Claude Code 或者其他 AI 助手），让它自动帮你搞定。

> **👉 请复制以下提示词并发送给 AI：**

```text
你好！我需要你帮我基于最新的 OpenClaw (v2026.3.12) 标准，安全地配置我的助手。请你一步步帮我执行以下操作：

1. 请先检查我的电脑上是否存在 `~/.openclaw/openclaw.json` 文件。如果没有，请帮我创建它。

2. 请帮我生成一个安全的网关 Token（运行 `openclaw doctor --generate-gateway-token` 获取，或者帮我生成长随机字符串）。

3. 请修改 `~/.openclaw/openclaw.json` 文件，严格应用以下"最小权限"安全基线：
   - 网关 (gateway)：只能本地访问 (`bind: "loopback"`)，开启 token 认证，并**开启防爆破限流**（`rateLimit`：1 分钟 10 次，锁定 5 分钟）。
   - 隐私隐藏 (discovery)：将局域网 mDNS 发现模式设为最小化 (`mdns: { mode: "minimal" }`)。
   - 会话 (session)：隔离所有人会话 (`dmScope: "per-channel-peer"`)。
   - 渠道控制 (channels)：所有社交软件的私聊策略设为 `pairing`（陌生人需配对），群聊必须艾特才能触发 (`requireMention: true`)。
   - 工具 (tools)：配置为 `messaging`，显式禁用高危工具 `["gateway", "cron", "sessions_spawn", "sessions_send", "whatsapp_login"]`。禁止直接执行 shell (`exec.security: "deny"`, `exec.ask: "always"`)，禁用提权。

4. 请修改配置以加固 Agent 隔离：
   - 开启全局 Docker 沙箱 (`sandbox.mode: "all"`)，并**彻底切断文件系统访问** (`sandbox.workspaceAccess: "none"`)。
   - 限制 AI 无限分身：设置 `subagents: { maxSpawnDepth: 1, maxChildrenPerAgent: 3 }`。

5. 配置完成后，请帮我运行 `openclaw security audit --deep` 深度检查安全状态，并把结果告诉我。如果有 critical 级别的严重警告，请用 `openclaw security audit --fix` 帮我修复。
```

*(提示：你现在就可以直接把上面这段话发给 AI，它立刻就会帮你自动配置好所有的最新安全机制！)*

## 🔍 第二步：这段提示词帮你做了什么？（白话解释）

这份 2026 版本的配置，比以前拉起了更严密的**六道防线**：

### 1️⃣ 防网络黑客（网关限制 & 防爆破）
不但强制只能你这台电脑本地访问，还加上了"密码输错 10 次就锁定 5 分钟"的防爆破机制。

### 2️⃣ 局域网隐身（mDNS 最小化）
你的机器人在同一 WiFi/局域网下会自动"隐身"，不会向别人广播它的敏感端口信息。

### 3️⃣ 防陌生人骚扰（渠道配对）
陌生人发私信，机器人绝对不会理，而是发给你一个"配对码"，你同意了它才会开始接待。群聊里也必须明确 `@` 它才会有反应。

### 4️⃣ 没收"核武器"工具（高危工具禁用）
彻底没收了机器人修改系统配置、设置定时任务的权限，也禁止了它跨账号乱发消息。

### 5️⃣ 关进"绝对黑屋"（无挂载沙箱）
以前是让它只能看指定文件夹（只读），现在是连看都不给看（`workspaceAccess: "none"`），纯粹作为一个独立大脑运行，彻底杜绝窃取本地文件的可能。

### 6️⃣ 禁止"无限套娃"（限制子代理）
防止 AI 脑抽，自己召唤无数个 AI 把你的电脑内存撑爆（最多只允许召唤一层、最多 3 个子代理）。

## 🆘 第三步：机器人"发疯"了怎么办？（小白救急手册）

如果机器人开始乱说话或不受控制，请像关掉漏水的水龙头一样，按顺序执行以下操作：

### 1️⃣ 强制关机：直接"掐断"它的气

这是最快、最有效的办法，不需要懂任何代码。

**如果你能看到一个黑乎乎、不停跳字的窗口：**
- 直接点击窗口右上角的 **[X]** 把它关掉。

**如果关不掉，就用"必杀技"：**
- 同时按下键盘上的 `Ctrl` + `Alt` + `Delete`（Mac 用户按 `Command` + `Option` + `Esc`）
- 在弹出的列表中找到名字为 **openclaw-gateway** 的进程
- 点击 **"结束任务"** 或 **"强制结束任务"**

---

### 2️⃣ 换把新锁：防止坏人继续搞破坏

如果你怀疑是别人在控制你的机器人，我们需要通过"终端"换个新密码（Token）。

**第一步：怎么打开那个黑窗口（终端）？**

- **Windows 用户**：点击屏幕左下角的"开始"图标，直接打字输入 `cmd`，然后按回车。
- **Mac 用户**：按住 `Command` + `空格键`，输入 `终端` 或 `Terminal`，然后按回车。

**第二步：输入"换锁"命令：**

在那个跳动的小光标后面，**完整复制**下面这段文字，然后按回车：

```bash
openclaw doctor --generate-gateway-token
```

> **注意**：看到屏幕显示出新的一串长代码（新 Token）后，旧的控制权就立刻失效了。

---

### 3️⃣ 终极断网：把它"关禁闭"

如果上面两步你觉得太复杂，直接在你的手机聊天软件上动手：

- **最快的一招**：直接在 Telegram 或 WhatsApp 里，把这个机器人账号**拉黑（Block）**。
- **断开连接**：打开你申请机器人时的对话框（比如 Telegram 的 `BotFather`），发送 `/revoke`。这相当于直接注销了机器人的"上网许可证"。
